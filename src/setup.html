Spotifort Setup Guide

Create your Spotify Developer app in 5 minutes

← Back to Spotifort

Why is this needed?

Spotify limits "Development Mode" apps to 5 users each. To work around this, each Spotifort user creates their own Spotify Developer app. You become the sole user of your own app, which means no limits.

Requirements: A Spotify Premium account. Free accounts cannot create Developer apps.

Step-by-Step Instructions

Step 1: Go to the Spotify Developer Dashboard

Open developer.spotify.com/dashboard in a new tab and log in with your Spotify account.

[Screenshot: Spotify Developer Dashboard login page]

Step 2: Create a new app

Click the Create app button in the top right corner.

[Screenshot: Dashboard with "Create app" button highlighted]

Step 3: Fill in the app details

Enter the following information:

  • App name: Anything you like (e.g., "My Spotifort", "Treefort Matcher")
  • App description: Anything (e.g., "Personal use")
  • Website: Leave blank or enter https://spotifort.com
  • Redirect URI: https://spotifort.com/callback

Important: The Redirect URI must be exactly https://spotifort.com/callback — no trailing slash, no typos. This is required for the authentication to work.

[Screenshot: Create app form with fields filled in]

Step 4: Select Web API and save

Under "Which API/SDKs are you planning to use?", check Web API.

Check the box agreeing to Spotify's Developer Terms of Service, then click Save.

[Screenshot: API selection and terms checkbox]

Step 5: Copy your Client ID

After creating the app, you'll be taken to your app's dashboard. Click Settings in the top right.

You'll see your Client ID — a 32-character string of letters and numbers. Click the copy button or select and copy it.

[Screenshot: App settings showing Client ID with copy button]

Step 6: Paste into Spotifort

Go back to Spotifort, paste your Client ID into the input field, and click Save & Continue.

That's it! You can now connect your Spotify account and find your Treefort artists.

Troubleshooting

"Invalid redirect URI" error

Double-check that your Redirect URI is exactly https://spotifort.com/callback. Common mistakes:

  • Trailing slash: https://spotifort.com/callback/ (wrong)
  • HTTP instead of HTTPS: http://spotifort.com/callback (wrong)
  • Typo in domain: https://spotifot.com/callback (wrong)

"Invalid Client ID" error

Make sure you copied the Client ID, not the Client Secret. The Client ID is 32 characters and is visible without clicking "View client secret".
